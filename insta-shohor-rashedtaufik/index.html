How Does Javascript Work?

JavaScript is an interpreted programming language. JavaScript on the client side works with the help of JavaScript engines that run behind the browsers. There are different engines written by different browser companies. Chakra for Internet Explorer, Firefox, V8 for Chrome etc. Now a day, the server side we use something called node.js . Which is an enhancement over the V8 engine and is used to run JavaScript for server side programming like creating API and talking to the databases.

What Does Javascript Event Loop Do?

We Know that the event loop is the JavaScript asynchronous Programming. JavaScript event loop is a process that waits for the Call Stack to be clear before pushing callbacks form the Task Queue to the call stack. Once the Stack is clear, the event loop triggers and check the Task Queue for available callbacks.
